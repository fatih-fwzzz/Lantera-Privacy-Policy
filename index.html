Persetujuan Pendamping

"Dengan ini, saya menyatakan bahwa saya telah membaca dan memahami penjelasan mengenai pengumpulan, penggunaan, dan penyimpanan data lansia yang saya dampingi. Saya memberikan persetujuan atas hal-hal berikut:"

  1. Pengumpulan Data Saya memahami dan menyetujui bahwa data milik lansia yang saya dampingi, termasuk hasil gula darah, tekanan darah, catatan makanan, konsumsi obat, keluhan kesehatan, serta hasil pemeriksaan laboratorium akan disimpan oleh aplikasi.
  2. Penggunaan dan Pemrosesan Data Saya memahami dan menyetujui bahwa data tersebut akan diproses untuk menghasilkan tren, pola, dan insight yang bertujuan meningkatkan pengalaman pemantauan dan memberikan rekomendasi yang relevan.
  3. Penyimpanan di Ekosistem Apple Saya mengetahui bahwa data akan disimpan dan diproses dalam ekosistem Apple, mengikuti standar keamanan dan perlindungan data yang berlaku.
  4. Persetujuan Tindakan Saya sebagai Pendamping Saya menyetujui bahwa seluruh data yang saya unggah atau isi untuk kepentingan lansia dilakukan atas sepengetahuan dan persetujuan lansia, dan menjadi tanggung jawab saya sebagai pendamping.
  5. Hak untuk Menarik Persetujuan Saya memahami bahwa saya dapat menarik persetujuan kapan saja dengan berhenti menggunakan aplikasi atau menghapus akun sesuai prosedur yang tersedia.

Persetujuan Lansia

"Dengan ini, saya menyatakan bahwa saya telah menerima penjelasan mengenai bagaimana data saya dikumpulkan, digunakan, dan disimpan oleh aplikasi. Saya setuju terhadap hal-hal berikut:"

  1. Pengumpulan Data Saya memahami dan menyetujui bahwa data diri dan data kesehatan saya, termasuk hasil gula darah, tekanan darah, makanan yang saya konsumsi, konsumsi obat, keluhan yang saya rasakan, serta hasil pemeriksaan laboratorium akan disimpan oleh aplikasi.
  2. Penggunaan dan Pemrosesan Data Saya memahami dan menyetujui bahwa data saya akan digunakan untuk menampilkan tren, pola, dan insight yang membantu saya dan pendamping memantau kondisi kesehatan saya.
  3. Penyimpanan di Ekosistem Apple Saya mengetahui bahwa data saya akan disimpan dan diproses dalam ekosistem Apple, sesuai standar keamanan dan perlindungan data yang berlaku.
  4. Pembagian Akses dengan Pendamping Saya menyetujui bahwa data saya dapat diakses oleh pendamping yang saya tunjuk, untuk membantu saya dalam pemantauan dan pengambilan keputusan.
  5. Hak untuk Menarik Persetujuan Saya memahami bahwa saya dapat menarik persetujuan kapan saja dengan berhenti menggunakan aplikasi atau meminta bantuan untuk menghapus akun sesuai prosedur.
